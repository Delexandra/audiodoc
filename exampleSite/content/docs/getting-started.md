## installation

download synthwave from the plugin manager or install manually:

1. download the `.vst3` or `.component` file from the releases page
2. copy it to your plugin directory:
   - **macOS:** `~/Library/Audio/Plug-Ins/VST3/`
   - **windows:** `C:\Program Files\Common Files\VST3\`
   - **linux:** `~/.vst3/`
3. rescan plugins in your DAW

{{< callout >}}
you don't need to configure anything to get started. load synthwave on a track, play a note, and you'll hear the default wavetable. start there and explore when you're ready.
{{< /callout >}}

## your first patch

1. load synthwave on a MIDI track
2. select the **"init"** preset from the preset browser
3. play some notes — you should hear a clean sawtooth wavetable
4. turn the **cutoff** knob to shape the tone
5. add some **reverb** from the effects section

## keyboard shortcuts

open the preset browser with {{< key "Ctrl" >}} + {{< key "P" >}} ({{< key "Cmd" >}} + {{< key "P" >}} on macOS).

toggle the arpeggiator with {{< key "Ctrl" >}} + {{< key "A" >}}.
